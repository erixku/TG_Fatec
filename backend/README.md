# HARPPIA - Back-end
Projeto de Trabalho de Graduação na FATEC Santana de Parnaíba, com o objetivo de desenvolver um sistema de gerenciamento de escalas das atividades e eventos do ministério de louvor da Igreja Evangélica Quadrangular Parque.
Nesta branch está toda a lógica de back-end, desenvolvida seguindo o esquema de monolito e a arquitetura Package By Feature.
